# unix-file-system


 * command	action *
 
 *  root   -	initialize root directory
 *  print  -	print current working directory and all descendants
 *  chdir  - change current working directory (.. refers to parent directory)
 *  mkdir  - sub-directory create 
 *  rmdir	 - delete 
 *  mvdir	 - rename 
 *  mkfil  - file create
 *  rmfil	 - delete
 *  mvfil	 - rename
 *  szfil	 - resize 
 *  exit   - quit the program
